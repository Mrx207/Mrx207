## On Debugging

Most React bugs I've fixed weren't obvious from reading the code. They required forming a hypothesis about runtime behavior, then verifying it — usually with React DevTools, targeted console instrumentation, and controlled reproduction steps. I find that process more interesting than writing greenfield code.
